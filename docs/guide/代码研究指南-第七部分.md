```

  │
  ├─→ tools/[name]/[name].ts (工具执行)
  │     ├─→ FileRead.ts
  │     ├─→ Bash.ts
  │     └─→ ... (43 tools)
  │
  ├─→ hooks/toolPermission/ (权限检查)
  │     ├─→ PermissionManager.ts
  │     ├─→ RuleMatcher.ts
  │     └─→ ContextValidator.ts
  │
  └─→ services/mcp/ (MCP 集成)
        ├─→ MCPClient.ts
        └─→ MCPServerManager.ts
```

### 组件依赖关系

```
App.tsx (根组件)
  ├─→ QueryInput.tsx
  │     └─→ useInput Hook (Ink)
  │
  ├─→ ResponseStream.tsx
  │     ├─→ useQueryEngine()
  │     └─→ useEffect() (事件订阅)
  │
  ├─→ ToolExecution.tsx
  │     └─→ useProgress()
  │
  └─→ StatusBar.tsx
        └─→ useSession()
```

### 数据流向图

**请求处理数据流：**
```
用户输入
  → QueryInput (组件)
    → QueryEngine.query()
      → LLM API 请求
        → 流式响应
          → token 事件
            → ResponseStream (组件)
              → UI 更新
```

**工具执行数据流：**
```
LLM 返回工具调用
  → QueryEngine.handleToolCalls()
    → PermissionManager.checkPermission()
      → 用户批准
        → Tool.execute()
          → 文件系统操作
            → 结果返回
              → QueryEngine.continueWithToolResults()
                → LLM 生成最终响应
```
